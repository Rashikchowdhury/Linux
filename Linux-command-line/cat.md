# To display content of a file in terminal

### Input:
<pre>

cat file_name
</pre>

## Example:

There is a file called textfile.txt.

now,

### input:

<pre>

cat textfile.txt
</pre>


### Output:

<pre>

a quick brown fox jumps over the lazy dog
my name is rashik
i am 2000 years old
abcs
i am a student
i read in class 1
</pre>
